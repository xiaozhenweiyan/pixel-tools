# 数学像素网站 Math Pixel Site

一个像素复古风数学学习网站，包含预测系统和计算器系统。

## 部署地址

https://xiaozhenweiyan.github.io/math-pixel-site/

## 功能列表

### 落地页
- 标题"数学学习网站"
- 卡片式入口：预测系统、计算机系统（可扩展）

### 预测系统
- **40 种数学预测方法**：基础统计、平滑法、回归、自回归、其他（含 10 种负数支持方法）
- **融合预测**：留一回测 MAPE 反权重融合，支持回测/均匀两种权重模式
- **神经网络预测（独立，不参与融合）**：
  - MLP 架构：动态输入层（2-8 神经元）、16 隐藏层（ReLU）、1 输出层
  - 渐进式训练：用前 k 个数字预测第 k+1 个，误差在容差内才训练下一组
  - 智能整数吸附：仅当预测值与最近整数差距 < 0.05 时才吸附
  - 自适应容差：整数 0.1、大数按比例、小数按精度
  - 数据增强：短序列生成多个训练样本
  - 多步预测差分修正：检测等差趋势避免发散
  - min-max 归一化（大数）+ z-score（小数）
  - 自适应学习率（patience 衰减）
  - 像素风网络可视化动画
- **过拟合算法（独立）**：最高次牛顿插值 + 高次多项式平均 + 三次样条插值，留一加权融合
- **偏移算法（独立）**：11 种简单函数族（常数、线性、二次、三次、绝对值、倒数、指数、幂、正弦、平方根、tanh）+ 水平/垂直平移，选 R² 最高的
- **长期训练模式**：像素风开关，累积序列持久化到 localStorage，神经网络增量训练（保留旧权重）
- **折线图**：拖拽平移、缩放、自制像素滚动条、悬停 tooltip
- **方法权重条形图**：40 种方法权重可视化
- **导出**：JSON / CSV

### 计算机系统
- 像素风计算器
- 三层布局：输出区 + 按键网格 + 长输入框
- 按键与输入框双向同步
- 支持 + - × ÷ 括号 小数 四则运算
- 安全表达式求值（不用 eval）

## 技术栈
- 纯 JavaScript（无框架、无构建工具）
- HTML5 Canvas（图表、网络可视化）
- CSS3（像素复古风、响应式）
- localStorage（长期训练持久化）
- GitHub Pages 部署

## 使用说明
1. 打开网站，看到"数学学习网站"落地页
2. 点击"预测系统"卡片进入预测页面
3. 在输入框输入数字序列（空格/逗号/换行分隔）
4. 选择预测数量、权重模式
5. 可选：开启"长期训练模式"做累积训练
6. 点击"预测"按钮
7. 查看融合结果、神经网络结果、过拟合、偏移算法、折线图、方法详情
8. 点击"返回首页"回到落地页
9. 点击"计算机系统"卡片进入计算器
10. 用按键或键盘输入表达式，按等号求值
