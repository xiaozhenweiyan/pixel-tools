# 像素工具大型更新 - Product Requirement Document

## Overview
- **Summary**: 本次更新包含五大模块：函数系统参数滑块增强、艺术类扩展（像素绘图编辑器 + 像素音乐合成器 + 生成器新算法 + 分组重构）、学习系统新增（数学学习卡片：四则运算 + 混合运算，多页动画展示）、移动端适配 + PWA 离线支持、中英文双语国际化。所有改动保持复古深空像素风格，统一三层页面结构。
- **Purpose**: 丰富网站功能矩阵，提升移动端体验，支持离线访问，形成学习/艺术两大分类的完整工具生态，并支持中英文双语自动切换。
- **Target Users**: 学生（数学学习）、像素艺术爱好者、音乐创作者、移动端用户、中外用户。

## Goals
- 函数系统支持参数化函数（a, b, c 等变量），滑块实时调整，支持动画往复
- 艺术类扩展为"像素图画"和"像素音乐"两个子分组，各含多个工具
- 新增像素绘图编辑器（画笔/橡皮/填色/调色板/撤销重做/导出）
- 新增像素音乐合成器（8-bit 音色 + 钢琴键盘 + 简单 sequencer）
- 像素艺术生成器新增 4 种算法（L-system 分形树 / Voronoi 镶嵌 / 波干涉 / 反应扩散）
- 学习系统新增"数学学习卡片"分组，含四则运算与混合运算两种卡片，多页动画展示
- 函数系统与计算器表达式安全加固
- 完整移动端响应式适配 + 触摸手势支持
- PWA 支持（manifest + service worker + 可安装 + 离线可用）
- 中英文双语国际化（自动检测系统语言 + 设置页手动切换）
- 更新 README.md 并同步到 GitHub

## Non-Goals (Out of Scope)
- 不引入构建工具（保持纯原生 HTML/CSS/JS）
- 不添加后端服务器（纯前端静态站点）
- 不做用户账号系统（保持 sessionStorage 临时账号）
- 不做单元测试框架

## Background & Context
当前项目是部署在 GitHub Pages 的复古像素风工具集合网站，已有功能：
- 学习类：像素数学（预测系统 / 函数系统 / 计算机系统）
- 艺术类：像素艺术生成器（4 种模式：流场 / 粒子 / 马赛克 / 螺旋）
- 个人系统：临时账号 + 头像 + 背景设置
- 三层页面结构：工具首页 → 分类首页 → 功能页

本次更新大幅扩展功能矩阵，同时提升移动端体验和离线可用性。

## Functional Requirements

### FR-1: 函数系统参数滑块
- 函数输入框下方新增参数滑块区域
- 自动检测函数表达式中的参数变量（a, b, c, d 等单字母变量，排除 x / sin / cos 等保留字）
- 每个滑块显示参数名、当前值、最小值、最大值、步长
- 每个滑块支持自定义最大/最小值和步长（点击设置图标弹出配置）
- 滑块拖动实时更新函数图像
- 支持动画模式：点击播放按钮后滑块在 min-max 之间往复运动，图像实时变化
- 动画速度可调

### FR-2: 首页艺术类重构为分组
- 艺术类分为"像素图画 PIXEL DRAWING"和"像素音乐 PIXEL MUSIC"两个子分组
- 像素图画分组下含：像素艺术生成器、像素绘图编辑器
- 像素音乐分组下含：像素音乐合成器
- 每个子分组有标题，卡片样式与现有一致
- 点击子分组卡片直接进入对应工具页面

### FR-3: 像素绘图编辑器
- 画布网格（默认 32×32，可调 16/32/64/128）
- 工具：画笔、橡皮、取色器、填色桶、直线、矩形、圆形
- 调色板：预设像素风 16 色调色板 + 自定义颜色选择器
- 撤销 / 重做（Ctrl+Z / Ctrl+Y，最多 50 步）
- 操作：左键绘画，右键取色
- 显示网格线开关
- 导出 PNG（原始像素尺寸，pixelated）
- 清空画布按钮
- 镜像绘画（水平/垂直，可选）

### FR-4: 像素音乐合成器
- Web Audio API 生成 8-bit chiptune 音色（方波/三角波/锯齿波/噪声）
- 钢琴键盘 UI：两个八度，鼠标点击发声
- 键盘按键映射（A-L / W-U 对应白键/黑键）
- 音色选择：方波 / 三角波 / 锯齿波 / 噪声
- 音量控制 + 八度切换
- 简单 Sequencer：16 步 × 4 轨网格编辑器，可播放循环
- BPM 可调（60-200）
- 波形可视化（示波器）

### FR-5: 像素艺术生成器新增算法
- 新增 4 种艺术模式：
  - L-system 分形树（递归分支，参数：深度、角度、长度衰减）
  - Voronoi 镶嵌（随机种子点 + 松弛算法，按细胞大小/距离配色）
  - 波干涉 Wave Interference（多波源叠加，振幅映射颜色）
  - 反应扩散 Reaction-Diffusion（Gray-Scott 模型，斑图生成）
- 每种模式有对应的参数滑块
- 种子化随机，可复现
- 与现有 4 种模式并列在下拉选择中

### FR-6: 函数系统与计算器安全加固
- 函数系统表达式解析从字符串替换升级为 AST 级解析
- 严格字符白名单 + 深度限制 + 执行超时保护
- 计算器表达式同样加固
- 输出安全审查报告

### FR-7: 学习系统首页 + 数学学习卡片分组
- 像素数学首页新增"学习系统"卡片（在预测系统 / 函数系统 / 计算机系统之后）
- 点击进入学习系统首页
- 学习系统首页有"数学学习卡片"分组（为未来扩展预留）
- 数学学习卡片下含：四则运算学习卡片、混合运算学习卡片
- 点击卡片进入对应学习页面

### FR-8: 四则运算学习卡片
- 多页（多关卡）方式展现：每页一道题，答对进入下一页
- 题型：加法、减法、乘法、除法（各 20 关，难度递增）
- 动画：题目出现动画、答对庆祝动画、答错提示动画
- 进度条显示当前关卡进度
- 输入答案后按回车/点击按钮提交
- 星级评价（3 星：全对；2 星：错 1-2 题；1 星：错 3+ 题）
- 每关完成后显示结果页（正确数、错题回顾、继续按钮）
- 像素风 UI，与整体风格一致

### FR-9: 混合运算学习卡片
- 多页方式展现，每页一道混合运算题
- 题型：加减混合、乘除混合、四则混合（带括号）
- 难度分级：简单（2 个数）→ 中等（3 个数）→ 困难（4 个数 + 括号）
- 动画效果与四则运算卡片一致
- 进度条 + 星级评价 + 结果页

### FR-10: 移动端响应式适配
- 所有页面在手机（320px+）和平板（768px+）上正常显示
- 布局自动调整：双栏变单栏、卡片纵向堆叠
- 字体大小自适应
- 触摸手势支持：
  - 函数系统：单指拖拽平移、双指捏合缩放
  - 像素绘图编辑器：手指绘画（适配压感或固定粗细）
  - 折线图：单指拖拽、双指缩放
- 按钮最小触控尺寸 44×44px
- 禁用双击缩放（viewport 设置）
- 移动端优化：虚拟键盘弹出时页面自适应

### FR-11: PWA 支持
- Web App Manifest（名称、图标、主题色、启动画面）
- Service Worker：缓存静态资源（HTML/CSS/JS/图片）
- 离线可用：已访问过的页面离线可打开
- 可安装到主屏幕（Add to Home Screen）
- 应用图标（192×192 和 512×512 PNG）
- 启动画面背景色与主题一致

### FR-12: 中英文双语国际化
- 创建翻译资源文件（i18n.js），包含中文和英文两套翻译
- 翻译范围：所有 UI 文字（标题、按钮、标签、提示、说明、菜单项等）
- 自动检测系统语言（navigator.language），中文用 zh，其余默认英文
- 设置页新增语言选择下拉框：跟随系统 / 中文 / 英文
- 语言选择持久化到 localStorage
- 切换语言时所有文字实时更新，无需刷新
- 标题（document.title）随语言切换
- HTML 中所有静态文字用 `data-i18n` 标记，JS 中动态文字用 i18n.t() 函数
- 函数输入、计算器输入等用户输入内容不翻译
- 代码中的错误提示、toast 消息也走翻译

### FR-13: README.md 更新
- 更新功能列表，涵盖所有新功能
- 更新页面结构图
- 更新文件结构
- 添加移动端和 PWA 说明
- 更新使用说明

## Non-Functional Requirements

- **NFR-1**: 性能 - 所有工具在主流浏览器中流畅运行（≥30fps 动画）
- **NFR-2**: 兼容性 - 支持 Chrome / Firefox / Safari 最新两个版本
- **NFR-3**: 安全 - 表达式求值无 XSS 风险，用户输入均经过滤
- **NFR-4**: 一致性 - 所有新功能保持复古深空像素风格
- **NFR-5**: 可访问性 - 主要交互支持键盘操作
- **NFR-6**: PWA 离线 - Service Worker 缓存后，断网可正常打开已访问页面

## Constraints
- **Technical**: 纯原生 HTML/CSS/JavaScript，无构建工具，无框架
- **Business**: GitHub Pages 静态托管，单页应用（所有页面在一个 index.html 内）
- **Dependencies**: p5.js（CDN）、Web Audio API、Canvas 2D、Service Worker API

## Assumptions
- 用户浏览器支持 Web Audio API 和 Service Worker
- 移动端为触屏设备
- PWA 图标使用像素风生成（无需设计稿）
- 学习卡片题目数量：四则运算每种 20 关，混合运算每级 15 关

## Acceptance Criteria

### AC-1: 函数系统参数滑块
- **Given**: 用户在函数系统输入含参数 a 的表达式（如 y=a*sin(x)）
- **When**: 参数区域显示 a 的滑块
- **Then**: 拖动滑块时函数图像实时更新
- **Verification**: `human-judgment`

### AC-2: 滑块动画往复
- **Given**: 参数滑块已存在
- **When**: 用户点击动画播放按钮
- **Then**: 滑块在 min-max 之间自动往复，图像实时变化
- **Verification**: `human-judgment`

### AC-3: 艺术类分组重构
- **Given**: 用户在工具首页
- **When**: 查看艺术类
- **Then**: 显示"像素图画"和"像素音乐"两个子分组，各含对应卡片
- **Verification**: `human-judgment`

### AC-4: 像素绘图编辑器基本功能
- **Given**: 用户在像素绘图编辑器页面
- **When**: 选择画笔工具在画布上绘画
- **Then**: 画布上出现对应颜色的像素点
- **Verification**: `human-judgment`

### AC-5: 像素音乐合成器发声
- **Given**: 用户在像素音乐合成器页面
- **When**: 点击钢琴键或按键盘按键
- **Then**: 发出对应音高的 8-bit 音色
- **Verification**: `human-judgment`

### AC-6: 生成器新算法
- **Given**: 用户在像素艺术生成器
- **When**: 选择新模式（分形树 / Voronoi / 波干涉 / 反应扩散）
- **Then**: 生成对应风格的像素艺术，种子可复现
- **Verification**: `human-judgment`

### AC-7: 学习系统卡片结构
- **Given**: 用户在像素数学首页
- **When**: 点击"学习系统"卡片
- **Then**: 进入学习系统首页，显示"数学学习卡片"分组及子卡片
- **Verification**: `human-judgment`

### AC-8: 四则运算学习卡片
- **Given**: 用户在四则运算学习页面
- **When**: 答对题目
- **Then**: 显示答对动画并进入下一题；答错显示提示
- **Verification**: `human-judgment`

### AC-9: 移动端适配
- **Given**: 用户用手机访问网站
- **When**: 浏览各页面
- **Then**: 布局自适应，按钮可点击，内容不溢出
- **Verification**: `human-judgment`

### AC-10: PWA 可安装
- **Given**: 用户首次访问网站
- **When**: 浏览器支持 PWA
- **Then**: 可将网站添加到主屏幕，作为应用打开
- **Verification**: `human-judgment`

### AC-11: README 更新
- **Given**: GitHub 仓库
- **When**: 查看 README.md
- **Then**: 包含所有新功能说明、更新的页面结构和文件结构
- **Verification**: `programmatic`

### AC-12: 安全加固
- **Given**: 函数系统和计算器
- **When**: 输入恶意表达式
- **Then**: 不执行恶意代码，返回错误提示
- **Verification**: `programmatic`

### AC-13: 双语自动检测
- **Given**: 用户首次访问网站
- **When**: 系统语言为中文（zh-CN / zh）
- **Then**: 界面显示中文；其他语言显示英文
- **Verification**: `human-judgment`

### AC-14: 语言切换
- **Given**: 用户在设置页
- **When**: 选择"跟随系统 / 中文 / 英文"
- **Then**: 界面文字实时切换，无需刷新
- **Verification**: `human-judgment`

### AC-15: 语言持久化
- **Given**: 用户选择了语言偏好
- **When**: 刷新页面
- **Then**: 保持上次选择的语言
- **Verification**: `programmatic`
