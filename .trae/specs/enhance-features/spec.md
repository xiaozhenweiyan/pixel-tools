# Pixel Tools 功能增强 - 产品需求文档

## Overview
- **Summary**: 为 pixel-tools 网站增强5个核心功能：鼠标拖拽特效、页面教程按钮、学习系统演示按钮修复、函数系统参数支持、坐标系标准长度显示
- **Purpose**: 提升用户体验和功能完整性，修复已知问题，增强交互性
- **Target Users**: 使用 pixel-tools 的所有用户

## Goals
- 实现鼠标拖拽粒子特效，确保在最顶层显示且不遮挡页面内容
- 为每个工具页面添加底部"教程"按钮，点击显示使用说明
- 修复学习系统演示按钮文字不可见问题
- 函数系统（2D/3D）全面支持参数：识别参数、滑动条、范围调节、步长、动画
- 坐标系缩放时显示统一标准长度，修复预测系统坐标系错乱

## Non-Goals (Out of Scope)
- 不修改现有工具的核心算法逻辑
- 不新增工具页面
- 不修改页面布局结构
- 不添加新的第三方依赖库

## Background & Context
- 当前项目使用纯前端技术（HTML/CSS/JS），基于 Canvas 2D 渲染
- 已删除背景粒子功能，现需恢复鼠标拖拽特效但避免遮挡问题
- 函数系统已有基本参数支持，但需要增强为完整的参数控制体系
- 坐标系使用 scale 属性进行缩放，但缺少单位长度显示

## Functional Requirements

### FR-1: 鼠标拖拽特效
- 在所有页面显示鼠标跟随的粒子拖拽效果
- 粒子特效使用最高 z-index，不遮挡交互元素
- 粒子使用 pointer-events: none，不干扰页面交互
- 切换页面时特效继续运行，不影响页面切换

### FR-2: 页面教程按钮
- 每个工具页面底部添加"教程"长按钮
- 点击按钮显示模态框，包含该页面的使用说明
- 模态框支持关闭（点击关闭按钮或点击遮罩）
- 教程内容包含：功能介绍、操作说明、快捷键（如有）

### FR-3: 学习系统演示按钮修复
- 修复四则运算、混合运算等学习卡片的"演示"按钮文字不可见问题
- 确保按钮文字颜色与背景对比度足够
- 按钮悬停和点击状态文字清晰可见

### FR-4: 函数系统参数支持
- 2D 和 3D 模式都支持参数识别（如 a, b, c 等）
- 每个参数显示滑动条控制当前值
- 支持调节参数的最小值、最大值
- 支持调节参数的步长
- 参数动画功能：自动循环变化参数值，速度可调

### FR-5: 坐标系标准长度
- 预测系统折线图和函数系统坐标系缩放时显示当前单位长度
- 单位长度随缩放自动调整（如 1, 0.5, 2, 0.25 等）
- 单位长度始终以合理精度显示（避免显示 1.333333）
- 修复预测系统坐标系缩放时标准长度错乱问题

## Non-Functional Requirements
- **NFR-1**: 性能：鼠标特效不影响页面流畅度（FPS ≥ 60）
- **NFR-2**: 响应式：教程按钮和模态框适配移动端
- **NFR-3**: 可访问性：按钮文字对比度符合 WCAG AA 标准
- **NFR-4**: 兼容性：所有功能在现代浏览器（Chrome/Firefox/Safari/Edge）正常工作

## Constraints
- **Technical**: 纯前端实现，不使用第三方库（除已有的 p5.js CDN）
- **Dependencies**: 基于现有代码结构，遵循现有模式
- **Timeline**: 所有功能一次性实现并验证

## Assumptions
- 用户使用现代浏览器（支持 ES6+）
- 用户了解基本的鼠标操作（拖拽、滚轮缩放）
- 教程内容使用中文（与现有 i18n 系统一致）

## Acceptance Criteria

### AC-1: 鼠标拖拽特效显示正常
- **Given**: 用户打开任意工具页面
- **When**: 用户移动鼠标
- **Then**: 看到跟随鼠标的粒子拖拽效果，点击页面元素正常响应
- **Verification**: `human-judgment`

### AC-2: 教程按钮功能正常
- **Given**: 用户在任意工具页面
- **When**: 用户点击底部"教程"按钮
- **Then**: 弹出模态框显示该页面的使用说明
- **Verification**: `human-judgment`

### AC-3: 演示按钮文字可见
- **Given**: 用户进入学习系统任意卡片页面
- **When**: 查看"演示"按钮
- **Then**: 按钮文字清晰可见，对比度足够
- **Verification**: `human-judgment`

### AC-4: 函数系统参数控制完整
- **Given**: 用户在函数系统页面输入含参数的函数（如 y=a*x+b）
- **When**: 使用参数滑动条、调节范围、设置步长、播放动画
- **Then**: 函数图像实时响应参数变化，动画平滑播放
- **Verification**: `human-judgment`

### AC-5: 坐标系单位长度显示正确
- **Given**: 用户在预测系统或函数系统缩放坐标系
- **When**: 放大或缩小视图
- **Then**: 坐标系显示当前单位长度，数值随缩放合理变化
- **Verification**: `human-judgment`
